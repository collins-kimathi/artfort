# Paintings Image Folder

Put painting images in this folder and reference them like:

`/paintings/your-image-file.jpg`

Example painting entry in `src/data/paintingsCatalog.js`:

```js
{
  id: 100,
  title: 'Morning Sky',
  artist: 'Your Name',
  size: '20 x 30 in',
  medium: 'Acrylic on canvas',
  year: '2026',
  collection: 'Landscape',
  price: 320,
  image: '/paintings/morning-sky.jpg',
}
```
